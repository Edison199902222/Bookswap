




单调栈一般寻找左右边界
当题目可以找出 在某个区域内， 某个target 可以作为最大值或者最小值 来贡献的话，可以考虑使用单调栈
lc 907， 有一个技巧， 就是找出左右边界后，如果我们要寻找 必须包含 target 并且要连续的子数组的组合时，
我们可以使用 （右边界 - target index ） * （target index - 左边界）
（右边界 - target index ）意思是 以target 为开头，可以找出 多少个 连续的子数组
（target index - 左边界）意思是 以target 为结尾，可以找出多少个 连续的子数组
然后 两个乘，相当于排列组合，就能求出 必须包含 target 并且要连续的子数组的组合
